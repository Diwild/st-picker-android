# ST选股工具 UI v2.0 更新说明

## 概述

全面优化了软件的UI设计，实现了更美观、专业、简洁的界面效果。

## 主要改进

### 1. 全新主题系统 (`ui/theme.py`)
- **现代化色彩方案**：深邃蓝黑色调，精心调配的对比度
- **玻璃拟态效果**：半透明背景、边框、高光
- **渐变支持**：主渐变、成功渐变、警告渐变、危险渐变
- **阴影系统**：轻度、中度、重度阴影层次
- **圆角规范**：统一的圆角半径系统

### 2. 动画系统 (`ui/animations.py`)
- **统一动画配置**：时长、缓动曲线常量
- **常用动画效果**：
  - 淡入淡出
  - 从底部滑入
  - 按压/释放反馈
  - 交错淡入（列表项依次出现）
  - 高度展开/收起

### 3. 股票卡片优化 (`ui/widgets/stock_card.py`)
- **阴影效果**：卡片底部阴影增加立体感
- **高光效果**：顶部细微高光增强质感
- **按压反馈**：点击时透明度变化
- **入场动画**：交错淡入效果
- **优化布局**：更合理的间距和字体大小

### 4. 阶段标签优化 (`ui/widgets/stage_badge.py`)
- **玻璃质感背景**：半透明色带边框
- **精致圆角**：6dp圆角更现代
- **优化尺寸**：根据文字自动调整

### 5. 评分条优化 (`ui/widgets/score_bar.py`)
- **高光效果**：进度条顶部高光增强质感
- **圆角轨道**：整体更柔和的视觉效果
- **动态颜色**：根据分数显示不同颜色

### 6. 筛选面板优化 (`ui/widgets/filter_panel.py`)
- **流畅动画**：展开/收起使用弹簧效果
- **子元素动画**：内容依次淡入
- **优化布局**：更合理的间距和字体
- **按钮样式**：统一的主题色按钮

### 7. 主屏幕优化 (`ui/screens/main_screen.py`)
- **视觉层次**：更好的背景、卡片层次
- **入场动画**：股票列表交错淡入
- **优化间距**：更舒适的视觉节奏
- **阴影效果**：卡片阴影增强立体感

### 8. 详情页优化 (`ui/screens/detail_screen.py`)
- **卡片阴影**：信息卡片带阴影效果
- **入场动画**：区域依次淡入
- **优化布局**：更清晰的视觉层次
- **圆角统一**：12dp圆角卡片

### 9. 主题常量更新 (`core/constants.py`)
- **新增颜色**：TEXT_TERTIARY、TEXT_HINT等
- **阶段颜色优化**：更鲜明的颜色区分
- **按钮色更新**：更现代的蓝色主按钮
- **圆角常量**：RADIUS_SMALL/MEDIUM/LARGE

## 设计特点

### 深色主题
- 主背景：#0D1015（深邃蓝黑）
- 卡片背景：#13171C（带层次感）
- 文字层次：白→浅灰→中灰→深灰

### 强调色
- 主蓝：#40A6FF（活力蓝）
- 成功绿：#40D980（清新绿）
- 警告橙：#FFA633（活力橙）
- 错误红：#FF5A5A（柔和红）

### 动画效果
- 交互动画：150ms快速反馈
- 过渡动画：250ms流畅过渡
- 缓动曲线：out_quad/out_back提供自然感

### 阴影系统
- 卡片阴影：轻微阴影增加立体感
- 按钮阴影：悬浮时更明显
- 层次分明：通过阴影区分层级
